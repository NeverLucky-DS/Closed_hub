## Что сделано

- **Лента:** реакции только справа сверху у карточки — чипы с эмодзи и счётчиком, «+» открывает выбор; комментарии убраны с UI и с API (таблица в БД остаётся).
- **Файлы:** у админа иконка корзины в **каждой строке списка** слева и в шапке просмотра — удаление встроено в вкладку «Файлы».
- **Новости:** у админа иконка «убрать из ленты» в правом верхнем углу карточки рядом с реакциями.
- **Веб-админ по умолчанию:** в `config` для `web_admin_telegram_ids` задан дефолт `1202549697`; явно пустая строка в `.env` (`WEB_ADMIN_TELEGRAM_IDS=`) отключает админов.
- **Вкладка «Хакатоны и мероприятия»:** создание команды (название, описание, даты, размер 2–30 с капитаном в счёте), список команд, карточка команды с **слотами как в мессенджерах** (аватар или пустой кружок), подача заявки, у капитана — принять/отклонить с превью профиля.

## Зачем

Упростить ленту, сделать модерацию и удаление файлов заметными внутри привычных вкладок, дать площадку для набора команд на хакатоны.

## Почему так

Один делегированный `click` на карточку реакций без повторной подписки после обновления DOM; команды и заявки — отдельные таблицы + простые POST с редиректом и `toast` в query.
