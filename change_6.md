# change_6

## Что сделано

- **`.gitignore` / `.dockerignore`:** шаблоны `closeml-*.json` и `*-service-account*.json`, чтобы ключи сервисного аккаунта не попадали в git и в образ при сборке.
- **Google Sheets / HR:** один **лист = одна компания** (название листа — имя компании из черновика, с санитизацией символов `[]:\/*?`). Если листа нет — создаётся. Строки дописываются **в следующую свободную строку**; колонки **A–D пустые**, заголовок и данные с **E по K** (время, UID HR, роль, вакансии, контекст, кто добавил, id в БД).

## Почему так

- Удобнее смотреть контакты по компаниям, без смешивания на одном листе.
- Пустые A–D оставлены под будущие поля или заметки вручную.

## Что вручную

- Старый лист `HR_contacts` (если был) в таблице можно удалить или оставить архивом — новый код его не использует.
- Если JSON уже случайно попал в git: `git rm --cached closeml-*.json` и закоммитить.
