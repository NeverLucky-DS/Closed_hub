# change_35

## Что добавлено

- **Обложка компании после создания:** POST `/companies/{slug}/photo` (заменяет фото, старые файлы на диске чистятся) и POST `/companies/{slug}/photo-clear` (сброс `photo_paths` + удаление файлов). Доступно автору карточки и веб-админу.
- При создании компании — **одно** поле фото вместо нескольких (в БД по-прежнему `photo_paths` как массив).
- Вспомогательные функции: `_cleanup_company_photos_disk`, фильтр шаблона `company_cover_class` для градиентных заглушек.

## Зачем

- Закрепить фото нужно и для уже существующих карточек, не только при создании.

## Почему так

- Переиспользованы те же правила, что и для фото при создании (`_prepare_company_photo_blobs`, `company_root`), плюс очистка «лишних» файлов в каталоге компании после замены.

## Дизайн

- Единый визуальный язык для списка компаний, хаба, подстраниц (HR / файлы / собесы), формы создания: градиентные акценты, карточки с обводкой, кнопки `btn` / `btn-primary`, общий блок `_company_subhead.html` с крошками и мини-обложкой.

## Что улучшить позже

- Редактирование названия и описания на хабе без пересоздания карточки.
- Ограничение размера/ресайз обложки на сервере.
