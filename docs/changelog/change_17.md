# change_17

## Google Sheets (HR)

- Колонки **только три**: Контакт · Дата и время (UTC) в формате **дд/мм/гггг ЧЧ:ММ UTC** · Комментарий (текст summary, до 30k символов).
- Убраны из выгрузки: роль, вакансии, TG id, id PostgreSQL (в **БД** `hr_contacts` по-прежнему всё нужное для черновика и LLM).
- **Строка 1** — заголовки, **строка 2** — пустая; первая запись с **строки 3**. После каждой записи добавляется **пустая строка**; следующая запись идёт через строку от предыдущей.
- Колонка комментария **шире** (~780 px), для столбца C включены **перенос строки** и выравнивание по верху.
- Данные пишутся в **A:C** (старые листы с блоком E:K не трогаются автоматически — при необходимости перенеси вручную или новый лист).

## БД

- Формат хранения **не менялся**: `TIMESTAMPTZ` в PostgreSQL остаётся нормальной практикой; «американский» вид в таблице убран за счёт **экспорта** в Sheets.
