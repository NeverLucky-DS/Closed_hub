# change_15

## Что сделано

- **`utils/nav_labels.py`** — единые подписи кнопок и **синонимы** для старых клавиатур в кэше Telegram (Справка, Узнать, Отмена и т.д.).
- **Главное меню:** одна строка «Собесы» + «Как пользоваться»; для whitelist — второй ряд «Пригласить человека». Убрана дублирующая вторая справка у не-whitelist.
- **Собесы:** «Читать по компаниям», «Рассказать про собес», «В главное меню»; в режиме рассказа — «Готово, сохранить» / «Выйти без сохранения».
- **Приглашение:** короткая клавиатура (отмена + подсказка), тексты мягче.
- **Тексты:** переписаны `/start`, справка (структура по блокам), HR-приглашение, подсказки в голосовом и interview flow.
- **Команда `/help`** — дублирует длинную подсказку для тех, кто ищет слэш-команды.

## Зачем

- Понятнее навигация и меньше «сухих» формулировок без потери смысла.
