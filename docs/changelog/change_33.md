## Что добавлено

- **Собес из бота → сайт:** после сохранения в markdown вызывается `publish_interview_to_site`: список компаний в Mistral (`company_match`, ключ `mistral_api_key_for_site` если задан), при совпадении — отзыв в `company_interview_reviews`, иначе создание компании и запись отзыва; дублирование файла в боте сохранено.
- **HR после «Верно»:** `link_confirmed_hr_to_company_line` — сначала точное сравнение `lower(trim(name))`, затем Mistral + при необходимости создание компании и `set_hr_contact_company`.
- **Файл в библиотеку:** после подтверждения папки второе сообщение с inline-кнопками компаний или «Пропустить» (`fco:` / `fcs:`).
- **Сайт:** вход в карточку компании — экран с тремя «папками» (HR, Собесы, Файлы) и отдельные URL `/companies/{slug}/hr|interviews|files`; привязка файла — выбор из своих подтверждённых файлов или ввод id.

## Зачем

Единая база компаний, меньше ручного ввода id, автоматическая маршрутизация контента из Telegram с опорой на список карточек на сайте.
