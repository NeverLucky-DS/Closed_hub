## Что сделано

- **`web/main.py`:** хост веб-сервера задаётся через **`WEB_HOST`** (по умолчанию `0.0.0.0` — уже было захардкожено; теперь можно переопределить, например `127.0.0.1` только для локального Mac).
- **`.env.example`:** комментарии и переменные **`WEB_HOST`**, **`WEB_PORT`** для доступа с телефона по Wi‑Fi.
- **`scripts/print-web-lan-url.sh`:** печатает готовый URL `http://<IP>:<порт>/` для телефона и напоминает про брандмауэр macOS.

## Зачем

Чтобы быстро открыть хаб с телефона в одной сети с Mac, без ручного поиска IP.

## Что сделать на своей машине

1. Запусти веб: `uv run python -m web.main` (или `docker compose up web`).
2. Выполни: `./scripts/print-web-lan-url.sh` — открой выведенный URL в браузере телефона.
3. Если не грузится — проверь, что телефон в той же Wi‑Fi, и при необходимости разреши входящие в брандмауэре macOS для Python.
