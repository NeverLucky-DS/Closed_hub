Нужно показать человеку в Telegram, что мы поняли из его рассказа о собеседовании, ПЕРЕД сохранением.

JSON с разбором (поля могут отсутствовать):
{extract_json}

Был ли ввод голосовым (расшифровка речи): {had_voice}

Ответь ТОЛЬКО JSON с ключами "preview_ru" и "summary_line" (оба строки).
preview_ru: 2–5 коротких строк обычным текстом (компания, позиция/уровень, когда). Без markdown (#, **, списков).
summary_line: одна строка без переносов; при had_voice=true — краткое резюме голоса (до ~120 символов); при только тексте можно "".
